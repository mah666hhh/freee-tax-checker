# freee取引入力 税務チェッカー Chrome拡張

freee経費入力時にリアルタイムで税務リスクを警告するChrome拡張機能です。

## 機能

### 経費チェック機能
- 経費登録時にClaude AIが自動で税務リスクを判定
- 🟢 白（問題なし）/ 🟡 グレー（要注意）/ 🔴 黒（高リスク）の3段階判定
- 摘要の改善サジェスト
- 調査で聞かれそうな質問を事前に把握
- 家事按分設定を考慮した判定
- 🟢判定時は自動登録オプション（確認画面をスキップして即登録）

### 勘定科目フィルター機能
- 取引一覧ページで特定の勘定科目の行を非表示にできる
- 年度レビュー時に「事業主貸（プライベート利用）」などのノイズを除去
- 右下の🔍ボタンからフィルターパネルを開く
- ページ内の勘定科目を自動スキャンしてリスト表示
- チェックを外した勘定科目の行が非表示になる
- ページネーション時も自動で再適用
- 設定は自動保存され、次回アクセス時も維持

## インストール方法

1. このフォルダをダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `freee-tax-checker` フォルダを選択

## 使い方

1. 拡張機能アイコンをクリックして設定画面を開く
2. APIキーと事業情報を入力して保存
3. freeeの取引登録画面（https://secure.freee.co.jp/deals）で経費を入力
4. 「支出を登録」ボタンをクリックすると自動でチェック
5. 結果に応じてモーダルが表示される

## 開発

```
freee-tax-checker/
├── manifest.json      # 拡張機能の設定
├── background.js      # Service Worker（Claude API呼び出し）
├── content.js         # 経費チェック機能
├── deal-filter.js     # 勘定科目フィルター機能
├── popup.html/js      # 設定画面
├── styles.css         # チェック結果モーダルのスタイル
├── deal-filter.css    # フィルターパネルのスタイル
└── icons/             # アイコン
```

## 注意事項

- このツールは税務アドバイスを提供するものではありません
- 最終的な判断は納税者ご自身、または税理士にご相談ください
- 情報提供のみを目的としています
