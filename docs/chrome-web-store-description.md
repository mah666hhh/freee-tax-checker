## 概要（短い説明）- 132文字以内

freee会計の取引入力画面で、経費登録前にAIが税務リスクを自動チェック。元国税調査官の視点で「調査で否認されないか」を判定します。

## 詳細な説明

freee取引入力 税務チェッカー

✨ 個人事業主版・法人版どちらのfreee会計にも対応！

freee会計の取引入力画面で、経費登録前にAIが税務リスクを自動チェックするChrome拡張機能です。

---

【AIリアルタイムチェック機能】
・経費の「登録」ボタンを押すと、AIが取引内容を即座に分析
・元国税調査官の視点で「この経費は調査で否認されないか？」を判定
・勘定科目、金額、摘要、取引先などの情報を総合的に評価
・事業情報や家事按分設定も考慮した精度の高いチェック

【分かりやすいリスク表示】
・🟢 問題なし：事業との関連が明確で、そのまま登録OK
・🟡 要注意：摘要の補足や按分の確認を推奨
・🔴 高リスク：税務調査で否認される可能性あり、要確認
・リスクレベル（1〜5）で詳細な危険度も表示

【改善サジェスト機能】
・「こう記録すれば通りやすい」という具体的な改善案を提示
・適切な摘要文の例をそのままコピーして使用可能
・「調査で聞かれそうな質問」を事前に把握できる

【勘定科目フィルター機能（Pro版）】
・取引一覧を勘定科目でワンクリック絞り込み
・複数科目の同時選択に対応
・経理作業の効率が大幅アップ
